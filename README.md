# Calculus Visual Learning System

**Calculus Visual** — веб-платформа для интерактивного изучения математического анализа: графики функций, производные, интегралы, пределы, уроки, тесты, чаты и инструменты для преподавателей.

| Параметр | Значение |
|----------|----------|
| Версия API | 2.0.0 |
| Python | 3.12+ |
| Лицензия | MIT |
| Документация API | http://localhost:8000/docs |
| Техническое задание (веб) | [docs/TZ.md](docs/TZ.md) |
| Техническое задание (мобильное) | [docs/TZ-mobile.md](docs/TZ-mobile.md) |

## Возможности

### Математические модули (доступны без входа, история — с входом)

| Модуль | URL | Описание |
|--------|-----|----------|
| Графики | `/graphs` | Построение одной или нескольких функций, пересечения, экспорт PNG, шаринг ссылки |
| Производные | `/derivatives` | Символьное дифференцирование (SymPy), график f и f′, касательная, анимация, шаги решения, PDF |
| Интегралы | `/integrals` | Определённый и неопределённый интеграл, визуализация площади, методы Трапеций и Симпсона |
| Пределы | `/limits` | Вычисление пределов, ряды Тейлора |
| Калькулятор | `/calculator` | Универсальный ввод выражений: значение, производная в точке, график |

### Обучение

- Уроки с контентом на **русском** и **таджикском**, фильтрация по категориям.
- Тесты с вопросами (один/несколько ответов), автоматическая проверка и сохранение результата.
- Прогресс студента (`/api/progress/me`).
- Система **достижений** (первый график, производные, тесты и т.д.).

### Роли

| Роль | Возможности |
|------|-------------|
| **Студент** | Уроки, тесты, чат класса, личные сообщения учителю/одноклассникам, профиль, уведомления |
| **Преподаватель** | Всё у студента + панель `/teacher`: классы, уроки, тесты, публикация, уведомления студентам, прогресс класса |
| **Администратор** | Панель `/admin`: пользователи, статистика, глобальное управление уроками и тестами |

### Коммуникация

- **Уведомления** в колокольчике (новые уроки/тесты при публикации).
- **Чат** (`/chat`): комнаты классов, личные диалоги, поиск контактов, WebSocket в реальном времени, индикатор «онлайн».

### Интерфейс

- Тема **светлая / тёмная**.
- Языки интерфейса: **RU**, **TG**, **EN** (файлы `app/static/locales/`).
- Адаптивная вёрстка (Bootstrap 5), PWA (кэш статики через Service Worker).

## Архитектура

```
┌─────────────────────────────────────────────────────────────┐
│                     Браузер (клиент)                        │
│  HTML (Jinja2) · Bootstrap · Chart.js · KaTeX · JS-модули   │
└──────────────────────────┬──────────────────────────────────┘
                           │ HTTP / WebSocket
┌──────────────────────────▼──────────────────────────────────┐
│                    FastAPI (main.py)                        │
│  Роутеры: auth, math, lessons, tests, teacher, admin,       │
│           chat, notifications, achievements, progress       │
└──────────────────────────┬──────────────────────────────────┘
                           │
        ┌──────────────────┼──────────────────┐
        ▼                  ▼                  ▼
   SQLAlchemy          SymPy/NumPy        JWT + bcrypt
   SQLite / PG         SciPy, ReportLab   SlowAPI (rate limit)
```

**Слои backend:**

- `app/routers/` — HTTP-эндпоинты.
- `app/services/` — бизнес-логика (математика, чат, уведомления, PDF, seed).
- `app/models/` — ORM-модели SQLAlchemy.
- `app/auth/` — JWT, зависимости `require_user`, `require_role`.
- `app/ws/` — менеджер WebSocket-соединений чата.

## Установка и запуск

### 1. Клонирование и виртуальное окружение

```bash
cd calculus-platform
python -m venv venv
```

**Windows (PowerShell):**

```powershell
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

**Linux / macOS:**

```bash
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Переменные окружения (опционально)

Скопируйте пример:

```bash
copy .env.example .env
```

Отредактируйте `.env` при необходимости (см. [Конфигурация](#конфигурация)).

### 3. Запуск сервера

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

При первом запуске:

- создаются таблицы БД;
- выполняются лёгкие миграции колонок;
- заполняются демо-пользователи, уроки, тесты и достижения.

### 4. Открыть в браузере

| Адрес | Назначение |
|-------|------------|
| http://localhost:8000 | Главная |
| http://localhost:8000/docs | Swagger UI |
| http://localhost:8000/redoc | ReDoc |

## Конфигурация

Файл `.env` (см. `.env.example`):

Создаются автоматически при первом старте (`app/services/seed_service.py`):

| Роль | Логин | Пароль | Email (демо) |
|------|-------|--------|----------------|

## API (краткий справочник)

Полная схема: **http://localhost:8000/docs**

### Аутентификация (`/api/auth`)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/register` | Регистрация |
| POST | `/login` | Вход (form: username, password) |
| GET | `/me` | Текущий пользователь |
| PATCH | `/preferences` | Язык, тема |
| POST | `/password-reset-request` | Запрос сброса пароля |
| POST | `/password-reset-confirm` | Подтверждение сброса |

### Математика (`/api/math`)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/plot` | График одной функции |
| POST | `/plot-multi` | Несколько функций |
| POST | `/derivative` | Производная + графики + шаги |
| POST | `/derivative/pdf` | PDF-отчёт |
| POST | `/tangent` | Касательная в точке |
| POST | `/integral` | Определённый интеграл |
| POST | `/integral/indefinite` | Неопределённый интеграл |
| POST | `/limit` | Предел |
| POST | `/taylor` | Ряд Тейлора |
| POST | `/check-derivative` | Проверка ответа студента |
| POST | `/evaluate-numeric` | Численное значение |
| POST | `/evaluate-function` | Значение функции в точке |
| POST | `/calculator/derivative-at` | Производная в точке |

На математические эндпоинты действует **rate limiting** (SlowAPI, обычно 20–30 запросов/мин).

### Обучение

| Префикс | Описание |
|---------|----------|
| `/api/lessons` | Публичные уроки (CRUD для авторов) |
| `/api/tests` | Тесты, прохождение, отправка ответов |
| `/api/progress/me` | Прогресс студента |
| `/api/achievements` | Список и «мои» достижения |

### Преподаватель (`/api/teacher`, роль teacher/admin)

- Классы: создание, зачисление студентов, прогресс.
- Уроки: CRUD, публикация/снятие, загрузка изображений.
- Тесты: CRUD, вопросы, импорт JSON, дублирование, публикация.

### Админ (`/api/admin`, роль admin)

- Пользователи, роли, активация.
- Статистика, глобальные уроки и тесты.

### Чат (`/api/chat`)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/contacts?q=` | Поиск собеседников |
| GET | `/rooms` | Список комнат |
| POST | `/direct/with/{user_id}` | Открыть личный чат |
| GET | `/rooms/{id}/messages` | История сообщений |
| POST | `/rooms/{id}/messages` | Отправить сообщение |
| WS | `/ws/{room_id}?token=` | WebSocket |

### Уведомления (`/api/notifications`)

- Список, отметка прочитанным.

## База данных и миграции

### Модели

- `User` — пользователи и роли.
- `Lesson`, `Test`, `TestQuestion`, `Result` — обучение.
- `ClassGroup`, `ClassEnrollment` — классы.
- `Achievement`, `UserAchievement` — геймификация.

### Автоматические миграции при старте

`app/database/migrate.py` — добавление новых колонок в существующую SQLite без Alembic.

### Alembic

```bash
alembic revision --autogenerate -m "описание изменения"
alembic upgrade head
```

Конфигурация: `alembic.ini`, `alembic/env.py`.

## Безопасность

- Пароли хешируются (**bcrypt**).
- API защищён **JWT** (роль в токене, проверка на эндпоинтах).
- **Валидация выражений** — блокировка опасных конструкций (`import`, `exec` и т.д.), лимит длины.
- **Rate limiting** на `/api/math/*`.
- CORS настроен для localhost (см. `main.py`).
- Сброс пароля через одноразовый токен (в dev-режиме ссылка может отображаться в ответе API).

## Разработка

### Добавление страницы

1. Создать `app/templates/my_page.html` (extends `base.html`).
2. Добавить маршрут в `PAGES` в `main.py`.
3. При необходимости — JS в `app/static/js/`.

### Добавление перевода

Ключи в `app/static/locales/ru.json`, `tg.json`, `en.json`. В HTML: `data-i18n="ключ"`.

### Формат математических выражений

Используйте синтаксис Python/SymPy:

- Степень: `x**2`
- Тригонометрия: `sin(x)`, `cos(x)`
- Экспонента: `exp(x)`
- Константа π: `pi`

### Чат: кто кому может писать

- **Студент** → учитель своего класса, одноклассники, админ.
- **Преподаватель** → любой студент, другие преподаватели, админ.
- **Админ** → все пользователи.

## Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Порт 8000 занят | `uvicorn main:app --port 8001` |
| Нет демо-пользователей | Удалить `calculus.db` и перезапустить, либо `POST /api/auth/init-demo` |
| График не рисуется | Проверить выражение; открыть консоль браузера; убедиться, что Chart.js загружен |
| Кнопка «Профиль» не видна | Войти в систему; обновить страницу (Ctrl+F5) |
| Чат пустой у студента | Преподаватель должен зачислить студента в класс (`/teacher` → классы) |
| Ошибка PostgreSQL | Проверить `DATABASE_URL`, права пользователя БД, `pip install psycopg2-binary` |
